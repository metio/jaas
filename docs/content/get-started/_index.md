Two ways to start: render Jsonnet locally with no cluster at all, or install the
JaaS service on Kubernetes and call its HTTP endpoint.

- **[Quickstart](/get-started/quickstart/)** — install JaaS and render your first
  snippet over HTTP.
- **[Install on Kubernetes](/get-started/kubernetes/)** — prerequisites and the
  Helm install in full.
- **[Local rendering](/get-started/local-rendering/)** — evaluate Jsonnet on your
  own machine, no cluster required.
