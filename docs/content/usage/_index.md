JaaS has two faces over one evaluation core: the **HTTP renderer** and the
**Flux operator** (`--enable-flux-integration`). Each page below configures one
feature of either face. For the exhaustive field-by-field detail, see the
[API reference](/api/jsonnetsnippet/).
